Fatos sobre o Stomatopoda

Os estomatópodes são predadores marinhos ativos que caçam presas com o auxílio de um sentido de visão muito mais aguçado, capaz de interpretar polarizações no espectro ultravioleta e infravermelho.

Classificação científica
Reino: Animalia
Filo: Arthropoda
Subfilo: Crustácea
Classe: Malacostraca
Subclasse: Hoplocarida
Ordem: Stomatopoda
